# Teoria dos Conjuntos

Este repositório contém a versão pública preliminar em PDF do livro **Teoria dos Conjuntos**, de Vinicius de Oliveira Rodrigues.

Este é um trabalho em andamento. O texto está incompleto e deve ser lido com cautela. O material é disponibilizado para leitura, estudo pessoal e uso didático não comercial.

## Arquivos

* [`settheory.pdf`](https://vo-rodrigues.github.io/settheory/settheory.pdf): versão pública mais recente do texto.


## Desenvolvimento

O desenvolvimento do livro ocorre em repositório privado.
Este repositório público serve apenas para disponibilização da versão compilada em PDF.
