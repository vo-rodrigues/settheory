\section{A sequĂŞncia dos alephs}\label{sec:sequenciaDosAlephs}
\index{aleph}
\index{sequĂŞncia dos alephs}
Na SeĂ§ĂŁo~\ref{sec:cardinaisENumeroDeHartogs}, vimos que, dado um cardinal $\kappa$, existe um cardinal $\kappa^+$ que ĂŠ o menor cardinal maior que $\kappa$.
AlĂŠm disso, temos que a uniĂŁo de uma coleĂ§ĂŁo de cardinais ĂŠ um cardinal.
AlĂŠm disso, $\omega$ ĂŠ o menor cardinal infinito.

Isso nos induz a definir, naturalmente, uma sequĂŞncia de cardinais:

\[
    \omega<\omega^+<\omega^{++}<\ldots
\]

Ao definir os primeiros $\omega$ termos dessa sequĂŞncia, temos que $\omega$, ĂŠ possĂ­vel definir o prĂłximo cardinal:
\[
    \bigcup\{\omega, \omega^+, \omega^{++}, \ldots\}
\]
A formalizaĂ§ĂŁo deste raciocĂ­nio nos leva Ă  sequĂŞncia dos alephs, que ĂŠ definida da seguinte forma:
\begin{definition}\label{def:sequenciaDosAlephs}\index{aleph}\index{alephs}
    Define-se, recursivamente, $\aleph_\alpha$ para cada ordinal $\alpha$ de modo que:
    
    \begin{itemize}
        \item $\aleph_0 = \omega$;
        \item $\aleph_{\alpha+1} = \aleph_\alpha^+$;
        \item $\aleph_\alpha = \sup\{\aleph_\beta : \beta < \alpha\}$, para $\alpha$ um ordinal limite.
    \end{itemize}
\end{definition}
Pelo Metateorema~\ref{mthm:esquemaDaRecursaoTransfinita}, tal feito ĂŠ possĂ­vel.

\begin{lemma}\label{lem:propriedadesBasicasDosAlephs}
    Para cada ordinal $\alpha$, $\aleph_\alpha$ ĂŠ um cardinal e, para todo $\beta<\alpha$, $\aleph_\beta < \aleph_\alpha$.
\end{lemma}
\begin{proof}
Procedemos por induĂ§ĂŁo transfinita.
Suponha que a afirmaĂ§ĂŁo seja verdadeira para todo $\beta<\alpha$.

Se $\alpha=0$, entĂŁo $\aleph_0 = \omega$ ĂŠ um cardinal, e a outra parte da afirmaĂ§ĂŁo ĂŠ trivial.

Para $\alpha = \beta + 1$, temos que $\aleph_\alpha = \aleph_\beta^+$ ĂŠ um cardinal.
Seja $\gamma < \alpha=\beta+1$.
Segue que $\gamma \leq \beta$.
Por hipĂłtese de induĂ§ĂŁo, $\aleph_\gamma \leq \aleph_\beta$.
Como $\aleph_\beta < \aleph_\beta^+ = \aleph_\alpha$, temos que $\aleph_\gamma < \aleph_\alpha$.

Se $\alpha$ ĂŠ um ordinal limite, entĂŁo $\aleph_\alpha = \sup\{\aleph_\beta : \beta < \alpha\}$ ĂŠ um cardinal, uma vez que ĂŠ a uniĂŁo de cardinais.
Seja $\gamma < \alpha$.
Como $\alpha$ ĂŠ limite, $\gamma+1<\alpha$.
Assim, $\aleph_\gamma < \aleph_{\gamma+1} \leq \aleph_\alpha$.
\end{proof}

A sequĂŞncia dos alephs ĂŠ, em verdade, a sequĂŞncia de todos os cardinais infinitos.
Antes, provaremos outro lema.

\begin{lemma}\label{lem:alephAlphaMaiorIgualAlpha}
    Para todo ordinal $\alpha$, $\aleph_\alpha\geq \alpha$.
\end{lemma}
\begin{proof}
Procedemos por induĂ§ĂŁo transfinita.
Suponha que a afirmaĂ§ĂŁo seja verdadeira para todo $\beta<\alpha$.
Se $\alpha=0$, entĂŁo $\aleph_0 = \omega \geq 0$.

Se $\alpha = \beta + 1$, entĂŁo $\aleph_\alpha = \aleph_\beta^+>\aleph_\beta \geq \beta$.
Assim, $\beta<\aleph_{\alpha}$, e, portanto, $\alpha=\beta+1 \leq \aleph_\alpha$.

Se $\alpha$ ĂŠ um ordinal limite, entĂŁo $\aleph_\alpha = \sup\{\aleph_\beta : \beta < \alpha\}\geq \sup\{\beta : \beta < \alpha\} = \alpha$.
\end{proof}
\begin{theorem}\label{thm:todoCardinalInfinitoEhAleph}\index{cardinal!infinito}
    Para cada cardinal infinito $\kappa$, existe um ordinal $\alpha$ tal que $\kappa = \aleph_\alpha$.
\end{theorem}
\begin{proof}
    Seja $\kappa$ um cardinal infinito.
    Assuma que, para cada ordinal $\alpha$, $\kappa \neq \aleph_\alpha$.

    Por induĂ§ĂŁo, veremos que, para todo ordinal $\alpha$, $\aleph_\alpha<\kappa$.
    Suponha que a hipĂłtese de induĂ§ĂŁo seja verdadeira para todo $\beta<\alpha$.

    Para $\alpha=0$, $\kappa \geq \omega = \aleph_0$, pois $\kappa$ ĂŠ infinito.
    Como $\kappa \neq \aleph_0$, segue que $\aleph_0<\kappa$.

    Suponha que $\alpha = \beta + 1$ para algum ordinal $\beta$.
    Como $\aleph_\beta < \kappa$ por hipĂłtese de induĂ§ĂŁo, $\aleph_\alpha=\aleph_\beta^+ \leq \kappa$, pois $\aleph_\beta^+$ ĂŠ o menor cardinal maior que $\aleph_\beta$.
    Por hipĂłtese de absurdo, $\kappa \neq \aleph_\alpha$, e, portanto, $\aleph_\alpha < \kappa$.

    Por fim, suponha que $\alpha$ ĂŠ um ordinal limite.
    Temos que, por hipĂłtese de induĂ§ĂŁo, $\aleph_\alpha=\sup\{\aleph_\beta : \beta < \alpha\} \leq \kappa$.
    Assim, $\aleph_\alpha < \kappa$

    Isso completa a induĂ§ĂŁo.
    Em particular, $\aleph_{\kappa+1} < \kappa$, o que ĂŠ absurdo, pois $\kappa <\kappa+1\leq \aleph_{\kappa+1}$.
\end{proof}

Outra notaĂ§ĂŁo para $\aleph_\alpha$ ĂŠ $\omega_\alpha$.
Na literatura, a alternĂ˘ncia das duas notaĂ§Ăľes ĂŠ bastante comum, e nĂŁo hĂĄ nenhum padrĂŁo estabelecido.
Neste texto, por motivos didĂĄticos, adotaremos a convenĂ§ĂŁo que, ao escrevermos $\omega_\alpha$, estamos pensando em $\aleph_\alpha$ como o conjunto $\aleph_\alpha$ munido de sua estrutura natural de boa-ordem, ou seja, enquanto ordinal.
JĂĄ ao escrevermos $\aleph_\alpha$, estamos pensando neste como conjunto de pontos sem nenhuma estrutura adicional, ou seja, como o conjunto representante canĂ´nico de sua cardinalidade.
Apesar deste artifĂ­cio didĂĄtico, muitas vezes tais intuiĂ§Ăľes se confundem, e utilizaremos as duas notaĂ§Ăľes de forma intercambiĂĄvel.

\begin{definition}
    Seja $\alpha$ um ordinal.
    Define-se $\omega_\alpha$ o ordinal $\aleph_\alpha$.
\end{definition}
