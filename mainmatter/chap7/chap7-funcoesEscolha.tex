\section{Funções Escolha}\label{sec:funcoesEscolha}
\index{função escolha|see{função de escolha}}\index{função de escolha}

Frequentemente, em matemática, temos uma família indexada $(A_i: i\in I)$ de conjuntos não vazios e queremos ``fixar um elemento $a_i \in A_i$ para cada $i\in I$''.
A Teoria dos Conjuntos, com os axiomas que temos até então, não é suficiente para garantir a existência de uma tal família $(a_i: i\in I)$ em qualquer tal situação.

Nesta seção, vamos estudar tais famílias.

\begin{definition}[$Z^--P$]\label{def:funcaoEscolha}\index{função de escolha}
    Seja $\mathcal C$ uma coleção de conjuntos não vazios.
    Uma \emph{função escolha} para $\mathcal C$ é uma função $c$ de domínio $\mathcal C$ tal que, para cada $a\in \mathcal C$, $c(a)\in a$.
\end{definition}
\begin{example}[$Z^--P$]
    Seja $\mathcal C=\{\{1, 2\}, \{3, 4\}\}$.

    Temos que $c:\mathcal C\to \{1, 2, 3, 4\}$ dada por

    \[
        c(a) = \begin{cases}
            1, & \text{se } a=\{1, 2\}\\
            3, & \text{se } a=\{3, 4\}\\
        \end{cases}
    \]
    é uma função escolha para $\mathcal C$.
\end{example}
Na Seção~\ref{sec:relacoesDeEquivalencia}, foi definida a noção de partição.
Lembremos que, nesse texto, uma partição de um conjunto $X$ é uma coleção $P$ de subconjuntos não vazios de $X$ dois-a-dois disjuntos e cuja união é $X$.

\begin{definition}[$Z^--P$]
    Seja $X$ um conjunto e $P$ uma partição de $X$.
    Um \emph{conjunto transversal} para $P$, ou \emph{conjunto de representantes} para $P$, é um subconjunto $T$ de $X$ tal que, para cada $a\in P$, $T\cap a$ é um conjunto unitário.
\end{definition}

\begin{example}[$Z^--P$]
    Seja $X=\{1, 2, 3, 4\}$ e $P=\{\{1, 2\}, \{3, 4\}\}$.
    Temos que $T=\{1, 3\}$ é um conjunto transversal para $P$, pois $T\cap \{1, 2\}=\{1\}$ e $T\cap \{3, 4\}=\{3\}$ são conjuntos unitários.
\end{example}


\begin{definition}[$Z^--P$]
    Seja $(A_i: i\in I)$ uma família indexada de conjuntos não vazios.
    Uma \emph{função escolha (indexada)} para $(A_i: i\in I)$ é uma função $c$ de domínio $I$ tal que, para cada $i\in I$, $c(i)\in A_i$.
\end{definition}
Novamente, notemos que se $A_i$ é vazio, não há nenhuma restrição sobre o valor de $c(i)$.

\begin{example}[$Z^--P$]
    Seja $I=\{1, 2\}$, $A_1=\{1, 2\}$ e $A_2=\{3, 4\}$.

    Temos que $c:I\to \{1, 2, 3, 4\}$ dada por
    \[
        c(i) = \begin{cases}
            2, & \text{se } i=1\\
            3, & \text{se } i=2\\
        \end{cases}
    \]
    é uma função escolha indexada para $(A_i: i\in I)$.
\end{example}

Suponha que exista uma função $f$ para $\mathcal P(A)$.
Intuitivamente, podemos, via recursão transfinita, bem-ordenar $A$:
vamos, um por vez, etiquetando os elementos de $A$ com ordinais $0, 1, 2, \dots \omega, \omega+1, \dots, \omega\cdot 2, \dots$, até exaurir os elementos de $A$.
Automaticamente, tal recursão nos dá uma bijeção de $A$ com um ordinal, o que induz, em $A$, uma boa-ordem.

A formalização deste raciocínio nos dá a seguinte proposição.

\begin{proposition}
    Seja $A$ um conjunto.
    São equivalentes:
    \begin{enumerate}[label=(\alph*)]
        \item Existe uma função escolha para $\mathcal P(A)$.
        \item $A$ é bem-ordenável.
    \end{enumerate}
\end{proposition}
\begin{proof}
    Caso $A=\emptyset$, o resultado é trivial.
    Vamos supor que $A\neq \emptyset$.
    Fixe $u \in A$ qualquer.
    (b) $\Rightarrow$ (a): caso $A$ seja bem-ordenável, seja $R$ uma boa-ordem em $A$.
    Defina $c:\mathcal P(A)\to A$ dada por:
    \[
        c(X) = \begin{cases}
            \min_R X, & \text{se } X\neq \emptyset\\
            u, & \text{caso contrário}
        \end{cases}
    \]
    É imediato que $c$ é uma função escolha para $\mathcal P(A)$.

    Reciprocamente, suponha que exista uma função escolha $c$ para $\mathcal P(A)$.
    Alterando-a se necessário, podemos supor que $c(\emptyset)\notin A$.
    Seja $b=c(\emptyset)$.

    Recursivamente, introduza um símbolo funcional $F:\ON \to A$ tal que, para cada ordinal $\alpha$,
    \[
        F(\alpha)=c\left(A\setminus \{F(\beta): \beta<\alpha\}\right).
    \]

    Existe um ordinal $\alpha$ tal que $F(\alpha)=b$.
    Do contrário, para todo ordinal $\alpha$, $F(\alpha)\in A\setminus\{F(\beta): \beta<\alpha\}$, de modo que $F$ é injetora.
    Em particular, $F|h(A): h(A)\to A$ é injetora, em que $h(A)$ é o número de Hartogs de $A$, um absurdo.

    Seja $\delta$ o menor ordinal tal que $F(\delta)=b$.
    Temos que $F| \delta: \delta \to A$ é uma função injetora pelo mesmo argumento acima, e $F(\delta)=b\notin A$, de modo que $A=\{F(\beta): \beta<\delta\}$.
    Ou seja, $F|\delta$ é uma bijeção de $\delta$ em $A$.

    Dessa forma, $A$ é bem-ordenável, pois a bijeção $F|\delta$ de $\delta$ em $A$ induz uma boa-ordem em $A$.
\end{proof}


Dessa forma, temos a seguinte proposição.
\begin{proposition}\label{prop:equivalenciaFuncaoEscolhaBoaOrdem}
     Sejam $A$ um conjunto.
     S
    São equivalentes:
    \begin{enumerate}[label=(\alph*)]
        \item Para todo conjunto $A$, existe uma função escolha para $\mathcal P(A)$.
        \item Todo conjunto é bem-ordenável.
    \end{enumerate}
\end{proposition}

A definição sobre boa ordenação, apesar de menos intuitiva, é mais simples: seu enunciado não depende do Axioma da Potência.
Por isso, neste texto, oficialmente, adotaremos ela como o Axioma da Escolha.

\begin{axiom}\label{ax:escolha}\index{axioma!da escolha}
    Todo conjunto é bem-ordenável.
\end{axiom}

Veremos agora algumas equivalências elementares do Axioma da Escolha.

\begin{proposition}
    Não assuma o Axioma da Escolha.
    São equivalentes:
    \begin{enumerate}[label=(\alph*)]
        \item O Axioma da Escolha.
        \item Para todo conjunto $A$, existe uma função escolha para $\mathcal P(A)$.
        \item Para toda família indexada $(A_i: i\in I)$ de conjuntos,existe uma função escolha indexada para $(A_i: i\in I)$.
        \item Seja $(A_i: i\in I)$ uma família indexada de conjuntos não vazios.
        Então $\prod_{i \in I} A_i \neq \emptyset$.
    \end{enumerate}
\end{proposition}
\begin{proof}
    (a) $\Leftrightarrow$ (b): devido à Proposição~\ref{prop:equivalenciaFuncaoEscolhaBoaOrdem}.

    (b) $\Rightarrow$ (c): Seja $I'=\{i\in I : A_i\neq \emptyset\}$.
    Seja $R$ uma boa ordem em $B=\bigcup_{i\in I'} A_i$.
    Seja $f':I'\rightarrow \bigcup_{i\in I'} A_i$ dada por $f'(i)=\min_R A_i$.
    Tome $b$ qualquer conjunto, e defina $f=f'\cup\{(i, b): i\in I\setminus I'\}$.
    Assim, $f:I\to \bigcup_{i\in I} A_i$ é uma função escolha indexada para $(A_i: i\in I)$.

    (c) $\Rightarrow$ (d): trivial.
    (d) $\Rightarrow$ (c): Seja $I=\mathcal P(A)\setminus \{\emptyset\}$ e $A_i=i$ para cada $i\in I$.
    Assim, $A_i\neq \emptyset$ para cada $i\in I$, e $\prod_{i\in I} A_i \neq \emptyset$ implica que existe uma função escolha indexada para $(A_i: i\in I)$.
    Temos que, para cada $i \in \mathcal P(A)\setminus \{\emptyset\}$, $f(i)\in A_i=i$.
    Seja $b$ qualquer conjunto e defina $g=f\cup \{(\emptyset, b)\}$.
    Então $g$ é uma função escolha para $\mathcal P(A)$.
\end{proof}
\subsection*{Exercícios}
\begin{exercise}
    Não assuma o Axioma da Escolha.
    Prove que o Axioma da Escolha é equivalente à seguinte afirmação.
    
    Para todo conjunto não vazio $X$ e toda partição $P$ de $X$ por conjuntos não vazios, existe um conjunto $Y\subseteq X$ tal que $Y\cap a$ é unitário para cada $a\in P$.
\end{exercise}
