O Exemplo~\ref{ex:soma-ordinal} justifica a definiĂ§ĂŁo de soma ordinal abaixo.
\begin{definition}\label{def:soma-ordinal}
    Sejam $\beta$ e $\alpha$ ordinais.
    Define-se a soma ordinal de $\beta$ e $\alpha$, denotada por $\beta+\alpha$, recursivamente em $\alpha$, de modo que:
    \begin{itemize}
        \item $\beta+0=\beta$;
        \item $\beta+S(\alpha)=S(\beta+\alpha)$;
        \item $\beta+\alpha=\bigcup\{\beta+\xi : \xi <\alpha\}$, se $\alpha$ ĂŠ um ordinal limite.
    \end{itemize}
\end{definition}

Note que tal definiĂ§ĂŁo ĂŠ uma extensĂŁo da soma de nĂşmeros naturais.
Observe que ela nĂŁo ĂŠ comutativa.

\begin{example}
    Temos que $\omega+1=\omega\cup\{\omega\}\neq \omega$.
    Por outro lado, $1+\omega=\sup\{1+n: n\in \omega\}=\omega$.
\end{example}


A subtraĂ§ĂŁo de ordinais tambĂŠm ĂŠ passĂ­vel de ser definida.

\begin{lemma}
    Sejam $\alpha, \beta$ ordinais.
    EntĂŁo $\alpha+\beta\geq \beta$.
\end{lemma}
\begin{proof}
    Procedemos por induĂ§ĂŁo em $\beta$.

    Para $\beta=0$, ĂŠ trivial.

    Suponha que $\alpha+\beta\geq \beta$ para um $\beta\in \ON$.
    EntĂŁo $\alpha+S(\beta)=S(\alpha+\beta)\geq S(\beta)$, pois $\alpha+\beta\geq \beta$.

    Finalmente, suponha que $\beta$ seja um ordinal limite e que $\alpha+\xi\geq \xi$ para todo $\xi<\beta$.
    EntĂŁo $\alpha+\beta=\bigcup\{\alpha+\xi : \xi <\beta\}\geq \bigcup\{\xi : \xi <\beta\}=\beta$.
\end{proof}

\begin{lemma}
    Sejam $\alpha, \beta, \gamma$ ordinais.
    Se $\beta<\gamma$, entĂŁo $\alpha+\beta<\alpha+\gamma$.
\end{lemma}
\begin{proof}
    Procedemos por induĂ§ĂŁo em $\gamma$.

    Para $\gamma=0$, a hipĂłtese $\beta<\gamma$ ĂŠ falsa, e, portanto, a tese ĂŠ verdadeira.

    Suponha que a tese seja verdadeira para um $\gamma\in \ON$.
    Vejamos que ĂŠ verdadeira para $S(\gamma)=\gamma+1$.

    Se $\beta<\gamma+1$, entĂŁo $\beta\leq \gamma$.
    
    Caso $\beta=\gamma$, entĂŁo $\alpha+\beta=\alpha+\gamma$.
    Caso $\beta<\gamma$, entĂŁo $\alpha+\beta<\alpha+\gamma$ pela hipĂłtese de induĂ§ĂŁo.
    De qualquer forma, $\alpha+\beta\leq \alpha+\gamma<(\alpha+\gamma)+1=\alpha+(\gamma+1)$.

    Finalmente, se $\gamma$ ĂŠ um ordinal limite e $\beta<\gamma$, entĂŁo $\beta\leq \xi$ para algum $\xi<\gamma$.
    Assim, $\alpha+\beta\leq \alpha+\xi<\alpha+(\xi+1)\leq \alpha+\gamma$.
\end{proof}

\begin{proposition}
    Para quaisquer ordinais $\beta$ e $\alpha$, se $\beta\leq \alpha$, entĂŁo existe um Ăşnico ordinal $\xi$ tal que $\beta+\xi=\alpha$.
    Tal ordinal ĂŠ denotado por $\alpha-\beta$.
\end{proposition}
\begin{proof}
    Para a existĂŞncia, dado $\alpha$, existe um ordinal $\xi$ tal que $\beta+\xi>\alpha$ (por exemplo, $\alpha+1$).
    Seja $\xi$ o menor tal ordinal.

    Temos que $\xi=0$ ĂŠ impossĂ­vel, pois $\beta+0=\beta\leq \alpha$.
    $\xi$ tambĂŠm nĂŁo pode ser limite, pois terĂ­amos $\beta+\xi=\bigcup\{\beta+\zeta : \zeta <\xi\}\leq \alpha$ pela minimalidade de $\xi$.

    Assim, $\xi=S(\gamma)$ para algum $\gamma\in \ON$.
    Temos que $\beta+\gamma\leq \alpha$ pela minimalidade de $\xi$.
    Se $\beta+\gamma<\alpha$, temos que $\beta+\xi=\beta+S(\gamma)=S(\beta+\gamma)\leq \alpha$, um absurdo.
    Assim, $\beta+\gamma=\alpha$.

    Para a unicidade, temos que, se $\gamma'$ fosse outro ordinal tal que $\beta+\gamma'=\alpha$, digamos que, sem perda de generalidade, $\gamma<\gamma'$.
    EntĂŁo temos $\alpha=\beta+\gamma<\beta+\gamma'=\alpha$, um absurdo.
\end{proof}

Uma interpretaĂ§ĂŁo da soma ordinal ĂŠ dada pela seguinte proposiĂ§ĂŁo.

\begin{proposition}
    Para quaisquer ordinais $\beta$ e $\alpha$, $\beta+\alpha$ ĂŠ o tipo da ordem de $C_{\beta, \alpha}=\{0\}\times \beta\cup\{1\}\times \alpha$, ordenado pela relaĂ§ĂŁo de ordem lexicogrĂĄfica.
\end{proposition}
\begin{proof}
    Procedemos por induĂ§ĂŁo em $\alpha$.
    Para $\alpha=0$, $\{0\}\times \beta\cup\{1\}\times 0=\{0\}\times \beta$ ĂŠ isomorfo a $\beta$ pela funĂ§ĂŁo $(0, \xi)\to \xi$.

    Suponha que a tese seja verdadeira para um $\alpha\in \ON$.
    EntĂŁo $C_{\beta, S(\alpha)}=C_{\beta, \alpha}\cup\{(1, S(\alpha))\}$, e $C_{\beta, \alpha}$ ĂŠ um segmento inicial de $C_{\beta, S(\alpha)}$.
    Seja $f: C_{\beta, \alpha}\to \beta+\alpha$ um isomorfismo.
    Estenda $f$ para $f': C_{\beta, S(\alpha)}\to \beta+S(\alpha)$ de modo que $f'((1, S(\alpha)))=\beta+\alpha$.
    EntĂŁo $f'$ ĂŠ um isomorfismo entre $C_{\beta, S(\alpha)}$ e $\beta+(\alpha+1)=(\beta+\alpha)+1$.

    Finalmente, suponha que $\alpha$ seja um ordinal limite e que a tese seja verdadeira para todo $\xi<\alpha$.
    EntĂŁo $C_{\beta, \alpha}=\bigcup_{\xi<\alpha} C_{\beta, \xi}$, cada $C_{\beta, \xi}$ ĂŠ isomorfo a $\beta+\xi$ e ĂŠ um segmento inicial de $C_{\beta, \alpha}$.

    Para cada $\xi<\alpha$, seja $f_\xi: C_{\beta, \xi}\to \beta+\xi$ o Ăşnico isomorfismo entre estas boas ordens.
    
    Pela unicidade do isomorfismo, para cada $\delta<\xi<\alpha$, $f_\xi|C_{\beta, \delta}=f_\delta$.
    Assim, $f=\bigcup_{\xi<\alpha} f_\xi: C_{\beta, \alpha}\to \bigcup_{\xi<\alpha} \beta+\xi=\beta+\alpha$ ĂŠ um isomorfismo.
\end{proof}

A multiplicaĂ§ĂŁo de ordinais ĂŠ definida como se segue.

\begin{definition}
    O produto de ordinais $\beta\cdot \alpha=\beta\alpha$ ĂŠ definido recursivamente em $\alpha$ por:
    \begin{itemize}
        \item $\beta \cdot 0=0$;
        \item $\beta \cdot S(\alpha)=\beta\cdot \alpha+\beta$;
        \item $\beta\cdot \alpha=\sup\{\beta\cdot \xi : \xi <\alpha\}$, se $\alpha$ ĂŠ um ordinal limite.
    \end{itemize}

    Tal definiĂ§ĂŁo se justifica como se segue:
    considere um operador binĂĄrio $G(\beta, s)$ que satisfaĂ§a, para todo ordinal $\beta$:
    \[
        G(\beta, s)=
        \begin{cases}
            0 & \text{se } s=\emptyset,\\
            s(\alpha)+\beta & \text{se } s\text{ ĂŠ funĂ§ĂŁo } \land  \exists \alpha \in \ON \dom(s)=\alpha+1,\\
            \bigcup_{\xi \in \alpha} s(\xi) & \text{se } s\text{ ĂŠ funĂ§ĂŁo } \land \exists \alpha \in \ON \dom(s)=\alpha \text{ ĂŠ limite}.
        \end{cases}
    \]
    O teorema da recursĂŁo nos dĂĄ, entĂŁo, uma receita para introduzir por definiĂ§ĂŁo $F(\beta, \alpha)=\beta\cdot \alpha$ tal que:
    \begin{itemize}
        \item $\beta\cdot 0=G(\beta, F_\beta|\emptyset)=0$;
        \item $\beta\cdot S(\alpha)=G(\beta, F_\beta|S(\alpha))=F_\beta(\alpha)+\beta=\beta\cdot \alpha+\beta$;
        \item $\beta\cdot \alpha=G(\beta, F_\beta|\alpha)=\bigcup_{\xi \in \alpha} F_\beta(\xi)=\sup\{\beta\cdot \xi : \xi <\alpha\}$, se $\alpha$ ĂŠ um ordinal limite.
    \end{itemize}
\end{definition}

Notemos que o produto de ordinais tambĂŠm nĂŁo ĂŠ comutativo: $2\cdot \omega=\sup\{2n: n\in \omega\}=\omega$, mas $\omega\cdot 2=\omega+\omega>\omega$.

A distributiva Ă  esquerda tambĂŠm nĂŁo ĂŠ vĂĄlida: $(1+1)\omega=2\omega=\omega$, mas $1\cdot \omega+1\cdot \omega=\omega+\omega>\omega$.

    A exponenciaĂ§ĂŁo de ordinais $\beta^\alpha$ ĂŠ definida recursivamente em $\alpha$ por:

    \end{itemize}

    Tal definiĂ§ĂŁo se justifica como se segue:

    O teorema da recursĂŁo nos dĂĄ, entĂŁo, uma receita para introduzir por definiĂ§ĂŁo $F(\beta, \alpha)=\beta^\alpha$ tal que:

\section*{ExercĂ­cios}
\begin{exercise}

\begin{exercise}
    Prove que, para todos $\alpha, \beta, \gamma \in \ON$:
    \begin{enumerate}[label=(\alph*)]

\end{exercise}

\begin{exercise}

    \begin{enumerate}[label=(\alph*)]

        \item Se $\alpha$ ĂŠ um ordinal limite e $\beta\neq 0$, entĂŁo $\beta\cdot \alpha$ ĂŠ um ordinal limite.

\end{exercise}
