\chapter{Conjuntos numéricos}\label{appendix:numbers}

No Capítulo~\ref{chap:numerosNaturais}, construímos os números naturais a partir dos axiomas de $Z^-$.
Neste capítulo, a partir dos mesmos axiomas, construiremos os números inteiros, os números racionais, os números reais e os números complexos.

Assumiremos o Axioma das Partes e o Axioma do Infinito durante todo o apêndice.
Porém, ressaltamos que:

\begin{itemize}
    \item Sem assumir o Axioma do Infinito nem o Axioma das Partes, é possível tratar, até certo ponto, os números inteiros e os números racionais enquanto classe de objetos.
    \item Sem assumir o Axioma das Partes, mas assumindo o do Infinito e o Esquema da Substituição, é plenamente possível construir os Inteiros e os Racionais basicamente com a mesma construção apresentada aqui.
\end{itemize}
Tais itens não serão tratados nesse texto pois não são importantes para o resto do livro.
